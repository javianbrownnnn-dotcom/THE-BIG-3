# The Speed Stack

Third-party tools for producing 30 videos in days without sacrificing quality,
mapped to the production pipeline (Scripting → Editing → Packaging →
Scheduled → Published). Prices checked July 2026 — verify before buying, and
start on free tiers where they exist.

**The golden rule:** tools accelerate a *system*, they don't replace one. Every
tool below plugs into a specific pipeline stage, and its output gets logged in
the video doc so the learning loop can tell you what's actually working.

## 1. AI voiceover (Scripting → Editing handoff)

The biggest unlock for narration-driven channels: lock the script in the video
doc, paste it into TTS, get broadcast-quality narration in minutes.

| Tool | Why | Rough price |
|---|---|---|
| **ElevenLabs** | The quality benchmark — natural pacing, emotional range from punctuation, best for documentary narration | Free tier; paid from ~$5/mo, scale plans higher |
| **Murf AI** | 200+ voices, 20 languages, easy per-video style tweaking | ~$19–29/mo |
| **PlayHT** | Voice cloning for a consistent channel voice across 30 videos | ~$29/mo range |
| **Amazon Polly** | Cheapest at scale (~$16/1M chars neural) if quality bar allows | Pay-per-character |

**Pipeline fit:** script sections in the video doc are written to be read
aloud — hook, body, outro. Export text → generate VO → log the VO file link in
the doc's asset tracker. One consistent voice per channel (clone it once).

## 2. AI / assisted editing (Editing stage)

| Tool | Why | Rough price |
|---|---|---|
| **Descript** | Text-based editing: edit the transcript, the cut follows. Ideal when VO drives the edit | ~$24/mo Creator |
| **CapCut (Pro)** | Fastest mobile-to-desktop workflow, huge template/effect library, generous free tier; captions need manual cleanup | Free; Pro ~$8/mo |
| **DaVinci Resolve** | Free, professional-grade; steeper learning curve but no ceiling | Free (Studio $295 once) |
| **AutoCut / Eddie AI class tools** | Auto b-roll placement and rough-cut assembly from a transcript — good first pass, human finishes | Varies, ~$10–30/mo |

**Pipeline fit:** the Retention Editing Pass SOP stays the quality gate — AI
does the assembly pass, a human does the retention pass. Log which workflow
each video used in the doc notes so the loop can compare retention by workflow.

## 3. Stock footage & visuals (Editing stage)

| Tool | Why | Rough price |
|---|---|---|
| **Storyblocks** | Unlimited downloads, broad library — the volume workhorse | ~$21–40/mo (annual) |
| **Artgrid** | Higher-end cinematic footage; universal license survives cancellation | ~$30–50/mo (annual) |
| **Pexels / Pixabay** | Free stock for filler b-roll | Free |
| **Midjourney** | AI stills for historical/abstract topics no stock library covers (ancient religion channel especially) | $10–30/mo |
| **AI b-roll generators** (Kapwing, OpusClip class) | Generate custom clips matched to transcript context | Subscription, varies |

**Pipeline fit:** the video doc's asset tracker lists needed b-roll per script
section before editing starts — batch-download in one session per video, not
mid-edit. For the religion channel, Midjourney-generated imagery + slow zooms
is a proven documentary style at near-zero cost.

## 4. Thumbnails & packaging (Packaging stage)

| Tool | Why | Rough price |
|---|---|---|
| **YouTube Test & Compare** (native) | A/B tests up to 3 thumbnails on real viewers, optimizes for watch time — free and the ground truth | Free |
| **Canva** | Fast thumbnail production with brand templates per channel | Free; Pro ~$13/mo |
| **Photopea** | Free Photoshop-class editor in the browser for finer control | Free |
| **ThumbnailTest.com** | Pre-publish testing of 10+ variants when native's 3-slot limit binds | ~$10–20/mo |
| **TubeBuddy** | A/B testing with statistical significance, plus SEO tooling | ~$10–30/mo |

**Pipeline fit:** thumbnail concept is written in the video doc at approval
time (per the packaging SOP); Canva template per channel makes production a
15-minute task; the native Test & Compare result gets logged back into the
doc's notes.

## Suggested starter stack for the 30-video sprint

Minimum viable spend (~$50–75/mo total):
1. **ElevenLabs** (Creator tier) — one cloned voice per channel
2. **CapCut Pro or DaVinci (free)** — whichever the editor prefers
3. **Storyblocks** + free Pexels — volume b-roll
4. **Canva Pro** + native Test & Compare — packaging

Batch pattern: write 5 scripts/day in the production workspace (AI drafts,
humans polish) → generate all 5 VOs in one session → edit in a batch → package
in a batch. The board's stages make the batching visible.
